# Interaction & Portfolio Designer

## Descripción

Este proyecto consiste en la configuración de un entorno de diseño y desarrollo para crear un portafolio web interactivo para estudiantes e investigadores.

El proyecto integra herramientas de inteligencia artificial, diseño UX/UI y desarrollo web, utilizando tecnologías como Figma, Claude Desktop, MCP, Antigravity 2.0 y Google Stitch.

## Objetivo general

Configurar el entorno inicial de trabajo, integrar herramientas inteligentes y documentar el proceso de desarrollo mediante un repositorio en GitHub y una bitácora de actividades.

## Herramientas utilizadas

- Git
- GitHub
- Node.js
- Figma
- Claude Desktop
- MCP
- Antigravity 2.0
- Google Stitch
